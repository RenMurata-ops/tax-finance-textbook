第1部：簿記の基本 ── 取引を記録する技術

この第1部では、事業のお金の動きを正確に記録するための技術「簿記」を、ゼロからすべて解説します。簿記は会計・税務・財務のすべての土台です。ここを理解しないまま先に進んでも、確定申告も決算も正しくできません。逆に言えば、簿記さえ理解すれば、あとの実務はすべてその応用にすぎません。

1-1. 簿記とは何か

簿記（ぼき）とは、事業で発生するすべてのお金の動きを、決められたルールに従って帳簿に記録する技術のことです。「簿記」という言葉は「帳簿に記入する」の略です。

なぜ簿記が必要なのか。理由は3つあります。

理由1：自分のビジネスの状態を正確に把握するため。記録がなければ、今月いくら儲かったのか、手元にいくらお金があるのか、借金がいくら残っているのかがわかりません。経営判断のしようがないということです。

理由2：税金を正しく計算するため。確定申告や法人税申告では、1年間の収入と経費を正確に集計する必要があります。帳簿がなければ、税金の計算ができません。

理由3：外部への説明のため。銀行から融資を受けるとき、取引先と契約するとき、投資家に説明するとき──すべて帳簿に基づいた財務諸表（ざいむしょひょう＝会社の成績表のようなもの）が求められます。

記録の方法には2種類あります。

単式簿記と複式簿記の違い

単式簿記（たんしきぼき）：家計簿と同じ方式です。「3月5日、文房具を買った、1,000円出ていった」のように、お金の出入りだけを1行で記録します。メリットは簡単なこと。デメリットは、「今、財産がいくらあるか」「借金がいくら残っているか」といった全体像が見えないことです。青色申告では10万円控除しか受けられません。

複式簿記（ふくしきぼき）：1つの取引を必ず「2つの側面」から記録する方法です。たとえば「文房具を現金で買った」なら、(1)「消耗品費という費用が1,000円増えた」と(2)「現金が1,000円減った」の両方を同時に記録します。すべての取引を借方（かりかた＝左側）と貸方（かしかた＝右側）の両方に同じ金額を記入するため、帳簿の左右は常に一致します。この仕組みのおかげで、記録ミスがあれば左右が合わなくなるので、自動的にエラーを検出できます。500年以上前にイタリアで考案された仕組みですが、現在でも世界中の企業が使っています。青色申告65万円控除を受けるにはこの方式が必須です。

1-2. 勘定科目の5分類

勘定科目（かんじょうかもく）とは、お金の動きに付ける「名札」のことです。たとえば「現金」「売上高」「地代家賃」などがすべて勘定科目です。取引が発生したとき、「この取引はどの名札を使って記録するか」を判断するのが、仕訳の第一歩です。

すべての勘定科目は、以下の5つのグループに分類されます。この5分類は簿記の最も基本的な枠組みなので、確実に理解してください。

分類意味具体的に何を指すか増加する側主要な勘定科目
資産会社が持っている財産現金、銀行預金、商品在庫、建物、土地、売掛金（まだ回収していない代金）など、「価値があるもの」すべて借方（左）現金、普通預金、当座預金、売掛金、受取手形、棚卸資産、前払費用、建物、車両運搬具、土地、ソフトウェア
負債将来支払う義務銀行からの借入金、仕入先にまだ払っていない代金（買掛金）、従業員から預かっている税金など、「いずれ出ていくお金」貸方（右）買掛金、支払手形、短期借入金、長期借入金、未払金、未払費用、預り金、前受金、未払法人税等
純資産資産から負債を引いた残り会社の「本当の持ち分」。オーナーが出したお金（資本金）と、これまで稼いで蓄積した利益（利益剰余金）の合計貸方（右）資本金、資本準備金、利益剰余金、繰越利益剰余金
収益事業で稼いだお金商品やサービスを提供して得た対価、預金の利息、資産の売却益など、「入ってきた価値」貸方（右）売上高、受取利息、受取配当金、雑収入、有価証券売却益、固定資産売却益
費用収益を得るために使ったお金商品の仕入代金、従業員の給料、事務所の家賃、電気代など、「事業を運営するためにかかったコスト」借方（左）仕入高、給料手当、法定福利費、地代家賃、水道光熱費、通信費、旅費交通費、消耗品費、減価償却費、支払利息、租税公課
借方・貸方の覚え方

なぜ資産と費用は「借方（左）で増える」のか。貸借対照表（BS）を思い浮かべてください。BSの左側には資産が並び、右側には負債と純資産が並びます。これが簿記のルールの原型です。

左側（借方）で増えるもの：資産と費用。「会社が持っているもの」と「使ったお金」は左。

右側（貸方）で増えるもの：負債、純資産、収益。「借りているお金」「自分の持ち分」「稼いだお金」は右。

減少するときは逆側に記入します。たとえば、現金（資産）が減るときは貸方（右）に記入します。

主要な勘定科目の詳細説明

よく使う勘定科目30選とその意味

資産の勘定科目

現金：紙幣・硬貨のこと。レジの中のお金や金庫のお金。

普通預金：銀行の普通預金口座の残高。

売掛金（うりかけきん）：商品やサービスを提供したが、まだ代金を受け取っていない状態。「ツケ」と同じ。後日、入金されたら売掛金が減って預金が増える。

棚卸資産（たなおろししさん）：まだ売れていない在庫商品のこと。期末に実際に数えて（棚卸して）金額を確定する。

前払費用：来期分の家賃や保険料を今期中に支払った場合、その「先払い分」を資産として計上する。来期になったら費用に振り替える。

建物・車両運搬具・土地：事業に使う固定資産。建物と車両は毎年少しずつ費用化（減価償却）するが、土地は価値が減らないので減価償却しない。

負債の勘定科目

買掛金（かいかけきん）：商品を仕入れたが、まだ代金を支払っていない状態。売掛金の逆。

未払金：商品仕入以外の債務。備品を購入して後払いにした場合など。

預り金：従業員の給与から天引きした所得税や住民税。会社のお金ではなく、従業員から「預かっている」お金なので負債。後日、税務署や市区町村に納付する。

前受金：商品を提供する前にもらった手付金や前払い金。まだサービスを提供していないので、売上にはできない。提供後に売上に振り替える。

収益の勘定科目

売上高：本業で稼いだ収入。最も基本的な収益科目。

受取利息：預金や貸付金から得られる利息収入。

雑収入：本業以外の小さな収入（自販機の設置手数料など）。

費用の勘定科目

仕入高：販売目的で商品を購入したときの代金。

給料手当：従業員に支払う給与・手当の総額（社会保険料や税金を天引きする前の金額）。

法定福利費：会社が負担する社会保険料（健康保険・厚生年金・雇用保険・労災保険の事業主負担分）。

地代家賃：事務所や店舗の賃料。

水道光熱費：電気代・ガス代・水道代。

通信費：電話代、インターネット回線費、切手代。

旅費交通費：電車賃、タクシー代、出張時の宿泊費。

消耗品費：文房具、コピー用紙、10万円未満の備品。

減価償却費：建物や車両などの固定資産を、耐用年数に応じて少しずつ費用化したもの。実際にお金が出ていくわけではないが、資産の価値減少を費用として認識する。

支払利息：借入金に対して支払う利息。

租税公課（そぜいこうか）：事業に関する税金（印紙税、固定資産税、自動車税など）。ただし所得税と法人税はここに含めない。

1-3. 仕訳の実務：よくある10パターン

仕訳（しわけ）とは、発生した取引を「借方（左）に何を、いくら」「貸方（右）に何を、いくら」と分解して記録する作業のことです。

仕訳の考え方は、次の3ステップです。

ステップ1：この取引で「何が増えた」「何が減った」かを考える。

ステップ2：増減したものが5分類のどれに当たるかを判断する（資産？負債？費用？収益？純資産？）。

ステップ3：増加したものを正しい側に記入する。資産・費用の増加は借方（左）、負債・純資産・収益の増加は貸方（右）。減少はその逆。

仕訳の思考プロセス（具体例）

「事務所の家賃15万円を銀行振込で支払った」

→ ステップ1：「家賃という費用が発生した」「普通預金が減った」

→ ステップ2：家賃＝費用、普通預金＝資産

→ ステップ3：費用の増加は借方（左）、資産の減少は貸方（右）

→ 仕訳：（借方）地代家賃 150,000 ／（貸方）普通預金 150,000

1-4. 帳簿の種類

主要簿と補助簿の一覧

主要簿（必ず作成するもの）：

(1) 仕訳帳：すべての取引を発生順に記録する帳簿。日付・勘定科目・金額・摘要を記入する。

(2) 総勘定元帳（そうかんじょうもとちょう）：仕訳帳の内容を勘定科目ごとに集計した帳簿。科目別の残高がわかる。

補助簿（必要に応じて作成）：

現金出納帳（現金の入出金記録）、預金出納帳、売掛金元帳（得意先別の売掛残高）、買掛金元帳、固定資産台帳（資産ごとの取得価額・減価償却を管理）、手形記入帳。

1-5. 試算表の作成ステップ

試算表（しさんひょう）を作る手順

ステップ1： 総勘定元帳の各勘定科目について、借方合計と貸方合計をそれぞれ集計する。

ステップ2： 合計試算表を作成する。全科目の借方合計と貸方合計を一覧にし、借方総合計＝貸方総合計になることを確認する（一致しなければ記帳ミスがある）。

ステップ3： 残高試算表を作成する。各科目の借方合計と貸方合計の差額（残高）を算出する。資産・費用は借方残高、負債・純資産・収益は貸方残高になる。

1-6. 決算整理仕訳

期末に帳簿の数字を実態に合わせるための修正仕訳です。これを行わないと正確な財務諸表が作れません。

第2部：財務諸表の作成実務

財務諸表（ざいむしょひょう）とは、会社の経営成績や財政状態をまとめた公式な書類です。「会社の健康診断書」と考えてください。主に3つの書類があり、これを財務三表と呼びます。損益計算書（PL）、貸借対照表（BS）、キャッシュ・フロー計算書（CF）です。この3つは互いにつながっており、一体で読むことで会社の全体像がわかります。

2-1. 損益計算書（PL）の作り方：5段階利益

損益計算書（そんえきけいさんしょ）は英語でProfit and Loss Statement、略してPLと呼びます。「一定期間（通常1年間）に、いくら稼いで、いくら使って、最終的にいくら残ったか」を示す書類です。

PLの最大のポイントは、利益を5段階に分けて計算することです。なぜ5段階に分けるかというと、「本業で儲かっているのか」「本業以外で損をしていないか」「突発的な損益はあったか」を区別して見るためです。

5段階利益の計算プロセス

(1) 売上総利益（うりあげそうりえき） = 売上高 - 売上原価

→ 「粗利（あらり）」とも呼ぶ。商品そのもので稼いだ利益。売上原価とは、売れた商品の仕入れ値のこと。

(2) 営業利益（えいぎょうりえき） = 売上総利益 - 販売費及び一般管理費

→ 本業で稼いだ利益。販管費には給料、家賃、広告費、減価償却費など、商品原価以外の事業運営コストがすべて含まれる。この数字がマイナスなら「本業で赤字」という深刻な状態。

(3) 経常利益（けいじょうりえき） = 営業利益 + 営業外収益 - 営業外費用

→ 本業＋金融活動を合わせた利益。営業外収益は受取利息や受取配当金。営業外費用は支払利息や為替差損。銀行からの借入が多い会社は、ここで利益が大きく減ることがある。

(4) 税引前当期純利益 = 経常利益 + 特別利益 - 特別損失

→ 毎年は起きない臨時の損益を加味した利益。特別利益は固定資産売却益など。特別損失は災害損失、リストラ費用など。

(5) 当期純利益（とうきじゅんりえき） = 税引前当期純利益 - 法人税等

→ 最終的に会社の手元に残る利益。これがBSの利益剰余金に加算され、会社の蓄えになる。

2-2. 貸借対照表（BS）の作り方

貸借対照表（たいしゃくたいしょうひょう）は英語でBalance Sheet、略してBSと呼びます。PLが「1年間の成績」なら、BSは「ある一瞬の財産の写真」です。決算日（たとえば3月31日）時点で、会社が「何を持っていて」「いくら借りていて」「差し引きの自分の取り分はいくらか」を一覧にした書類です。

BSの最も重要なルールは「資産 = 負債 + 純資産」という等式が必ず成り立つことです。左側（資産）と右側（負債＋純資産）が必ず同じ金額になるので「バランスシート」と呼ばれます。

資産と負債はそれぞれ「流動」と「固定」に区分します。流動とは「1年以内に現金化できる（または支払期限が来る）もの」、固定とは「1年を超えて使い続ける（または支払期限が1年超先の）もの」です。この区分は、会社の短期的な支払い能力を判断するために重要です。

資産の部（左側）負債・純資産の部（右側）
流動資産（1年以内に現金化）流動負債（1年以内に支払）
現金・預金、売掛金、棚卸資産買掛金、短期借入金、未払金
固定資産（1年超使用）固定負債（1年超後に支払）
建物、車両、土地、ソフトウェア長期借入金、社債
純資産
資本金、利益剰余金

2-3. キャッシュ・フロー計算書（CF）

直接法と間接法の違い

直接法：営業活動の現金収入・支出を直接集計する方法。わかりやすいが手間がかかる。

間接法（実務で主流）：税引前当期純利益から出発し、非資金項目（減価償却費など）や運転資本の増減を加減して営業CFを算出する。

間接法の計算式：営業CF = 税引前当期純利益 + 減価償却費 - 売掛金増加 + 買掛金増加 - 棚卸資産増加 ...

第3部：税務申告の実務

税務申告とは、「自分の所得（もうけ）がいくらだったかを計算して、税務署に届け出て、税金を納める」一連の手続きのことです。日本の税制は「申告納税制度」を採用しており、これは「国が税額を決めて通知する」のではなく、「納税者自身が計算して申告する」という仕組みです。つまり、正しく計算する責任は自分にあります。間違えれば追加で税金を取られ（加算税）、悪質な場合は罰金（重加算税）や刑事罰もあります。

この第3部では、個人事業主と法人のそれぞれについて、実務で必要な申告手続きをすべて解説します。

3-1. 確定申告の手順

確定申告の流れ（個人事業主の場合）

対象期間：1月1日～12月31日の所得について、翌年2月16日～3月15日に申告。

ステップ1： 1年間の帳簿を締め、収入金額と必要経費を集計する。

ステップ2： 必要書類を準備する。確定申告書、青色申告決算書（または収支内訳書）、源泉徴収票、控除証明書（生命保険・社会保険など）、マイナンバーカード。

ステップ3： 申告書を作成する。国税庁「確定申告書等作成コーナー」またはe-Taxソフトで入力。

ステップ4： 提出する。e-Tax（電子申告）、郵送、または税務署窓口。e-TaxならマイナンバーカードとスマートフォンまたはICカードリーダーが必要。

ステップ5： 納税する。口座振替、クレジットカード、コンビニ、またはQRコード決済。振替納税の場合、口座引落は4月中旬。

3-2. 青色申告

青色申告65万円控除の要件（2025-2026年）

(1) 事前に「青色申告承認申請書」を税務署に提出（開業から2か月以内、または適用年の3月15日まで）。

(2) 複式簿記で記帳すること。

(3) 貸借対照表と損益計算書を確定申告書に添付すること。

(4) e-Taxで提出、または電子帳簿保存を行うこと（これがないと控除額は55万円に減額）。

(5) 期限内に申告すること（期限後申告は10万円控除に減額）。

3-3. 消費税申告

消費税は、商品やサービスを買うときに消費者が支払う税金（税率10%、食品は8%）ですが、実際に国に納めるのは事業者です。事業者は「お客さんから預かった消費税」から「仕入先に支払った消費税」を差し引いた差額を国に納付します。

課税事業者の判定：すべての事業者が消費税を申告するわけではありません。以下の条件に該当する場合に「課税事業者」となり、消費税の申告・納付義務が生じます。

(1) 基準期間（個人：2年前、法人：2事業年度前）の課税売上高が1,000万円超。

(2) または、特定期間（個人：前年1月〜6月、法人：前事業年度の上半期）の課税売上高が1,000万円超。

(3) または、インボイス発行事業者として登録した場合（売上が1,000万円以下でも登録すれば課税事業者になる）。

項目本則課税簡易課税
計算方法預かった消費税 - 支払った消費税預かった消費税 × (1 - みなし仕入率)
適用条件制限なし基準期間の課税売上高5,000万円以下 + 届出
メリット実際の経費が多い場合に有利計算が簡単、仕入が少ない業種に有利
インボイス制度（2023年10月～）

適格請求書（インボイス）を発行できるのは登録を受けた課税事業者のみ。免税事業者からの仕入は原則として仕入税額控除不可（経過措置：2026年9月まで80%控除可、2029年9月まで50%控除可）。登録番号は「T + 法人番号（13桁）」または「T + 13桁の番号（個人）」。

3-4. 法人税申告の基礎

法人税の計算プロセスと主要別表

計算の流れ：会計上の利益（税引前当期純利益） → 税務上の加算・減算調整 → 課税所得 → 税率適用 → 法人税額

2025-2026年の法人税率：資本金1億円以下の中小法人は、年800万円以下の部分が15%、800万円超の部分が23.2%。

主要な別表：

別表一：法人税額の計算 / 別表四：所得の金額の計算（加算・減算） / 別表五(一)：利益積立金の計算 / 別表五(二)：租税公課の納付状況 / 別表十五：交際費の損金算入限度額

申告期限：事業年度終了日の翌日から2か月以内。届出により1か月延長可。

3-5. 源泉徴収と年末調整

源泉徴収とは、給与や報酬を支払う際に、所得税を天引き（差し引いて）して、会社が本人に代わって国に納付する制度です。

報酬（士業等）10.21%（100万円超は20.42%）翌月10日

第4部：経理実務フロー

経理の仕事は「日次」「月次」「年次」の3つのサイクルで回っています。日次業務は毎日の記録、月次業務は1か月ごとの集計と確認、年次業務は1年間の総まとめです。この3つのサイクルを確実にこなすことが、正確な決算と税務申告の基盤になります。

以下、それぞれのサイクルで「何を」「なぜ」「どうやるか」を解説します。

4-1. 日次業務

(1) 現金管理：毎日の営業終了後、手元現金を実際に数え（実査）、帳簿残高と一致するか確認する。差異があれば「現金過不足」勘定で処理。

(2) 領収書・請求書の整理：受領した領収書に日付・金額・取引先・内容を確認し、ファイリングする。電子帳簿保存法の要件を満たすスキャン保存も可。

(3) 仕訳入力：当日発生した取引を会計ソフトに入力する。後回しにするとミスが増えるため、当日中が理想。

4-2. 月次業務

(1) 月次決算：月末に試算表を作成し、売上・経費の推移を確認する。予算との差異分析を行い、経営者に報告する。

(2) 残高照合：預金通帳と帳簿の残高を照合する。売掛金・買掛金の残高を取引先と確認する。

(3) 給与計算：勤怠データを集計 → 総支給額を計算 → 社会保険料・所得税・住民税を控除 → 差引支給額を算出 → 振込処理。

(4) 源泉所得税の納付：翌月10日まで（納期の特例を受けている場合は年2回）。

4-3. 年次業務

決算スケジュール（3月決算の場合）：

時期作業内容
3月末期末棚卸の実施、固定資産の実査
4月上旬決算整理仕訳の入力、経過勘定の計上
4月中旬財務諸表の作成、税理士との確認
5月末法人税・消費税・地方税の申告と納付

第5部：実務の財務分析

財務分析とは、財務諸表の数字を使って「この会社は健全か」「儲かっているか」「お金の使い方は効率的か」を判断する技術です。数字を見るだけでは意味がわかりません。数字を「指標（しひょう）」に変換し、業界平均や過去の数値と比較することで、初めて意味のある情報になります。

財務分析は大きく4つの視点で行います。収益性（どれだけ効率よく稼いでいるか）、安全性（借金を返す力があるか）、効率性（資産をうまく活用しているか）、成長性（売上や利益が伸びているか）です。

5-1. 収益性分析

ROEデュポン分解

ROE（自己資本利益率）は3つの要素に分解して原因分析ができます。

ROE = 純利益率 × 総資産回転率 × 財務レバレッジ

= (当期純利益/売上高) × (売上高/総資産) × (総資産/自己資本)

数値例：純利益4,060千円 / 売上50,000千円 × 売上50,000千円 / 総資産30,000千円 × 総資産30,000千円 / 自己資本15,000千円

= 8.12% × 1.67回 × 2.0倍 = ROE 27.1%

営業利益率 = 営業利益 / 売上高 × 100。業種平均と比較する。上記例では6,000/50,000 = 12.0%。

ROIC（投下資本利益率）= NOPAT（税引後営業利益）/ 投下資本。株主と債権者の両方から見た効率性を測る指標。WACC（加重平均資本コスト）を上回っていれば価値創造している。

5-2. 安全性分析

安全性分析は「この会社は借金を返せるか」「突然の支払いに耐えられるか」を判断する分析です。いくら利益が出ていても、短期的な支払い能力がなければ倒産します。

指標計算式目安数値例
流動比率流動資産 / 流動負債 × 100200%以上が理想18,000/5,000 = 360%
当座比率当座資産 / 流動負債 × 100100%以上14,000/5,000 = 280%
自己資本比率自己資本 / 総資産 × 10040%以上15,000/30,000 = 50%
D/Eレシオ有利子負債 / 自己資本1.0倍以下10,000/15,000 = 0.67倍

5-3. 効率性分析と資金繰り

CCC（キャッシュ・コンバージョン・サイクル）＝商品を仕入れてから、販売して現金を回収するまでの日数。短いほど資金効率が良い。

CCC計算例

売上債権回転日数 = 売掛金6,000 / (売上50,000/365) = 43.8日

棚卸資産回転日数 = 棚卸資産4,000 / (売上原価30,000/365) = 48.7日

仕入債務回転日数 = 買掛金5,000 / (売上原価30,000/365) = 60.8日

CCC = 43.8 + 48.7 - 60.8 = 31.7日

資金繰り表の作成：月ごとに「前月繰越 + 当月入金 - 当月出金 = 翌月繰越」を一覧にした表。経常収支（売上入金-仕入支払-経費）と経常外収支（借入-返済）に区分して作成する。月末残高がマイナスになる月があれば、事前に融資や入金促進の対策が必要。
