# 第1部：戦略会計──数字を経営判断の武器にする

---

## 第1章 戦略会計とは何か

### 1-1. 伝統的会計と戦略会計の違い

企業会計には大きく分けて二つの役割がある。一つは過去の取引を正確に記録し、外部のステークホルダーに報告する「財務会計」であり、もう一つは経営者の意思決定を支援する「管理会計」である。戦略会計（Strategic Management Accounting）とは、この管理会計をさらに発展させ、企業の競争戦略と密接に結びつけた会計手法の総称である。

伝統的な管理会計が工場内のコスト管理や予算統制に重点を置くのに対し、戦略会計は市場環境、競合他社の動向、バリューチェーン全体のコスト構造、さらには顧客の生涯価値までを分析対象とする。つまり、社内の数字だけでなく、社外の情報をも取り込んで経営判断の材料にする点が決定的に異なる。

| 項目 | 伝統的管理会計 | 戦略会計 |
|------|--------------|---------|
| 時間軸 | 過去・現在 | 現在・未来 |
| 視点 | 社内中心 | 社内＋社外（競合・市場） |
| 目的 | コスト管理・予算統制 | 競争優位の構築 |
| 対象期間 | 短期（月次・年次） | 中長期（3〜10年） |
| 分析単位 | 部門・製品 | バリューチェーン・事業ポートフォリオ |

### 1-2. なぜCFOに戦略的思考が求められるのか

現代のCFO（最高財務責任者）に求められる役割は、かつてとは大きく異なっている。従来のCFOは「数字の番人」として、正確な帳簿の維持と税務申告、そして資金繰りの管理が主な職務であった。しかし、グローバル競争の激化、テクノロジーの急速な進化、そして不確実性の高まりにより、CFOはCEOの戦略パートナーとしての役割を担うようになっている。

経済産業省が公表した「企業価値向上に向けた経営リーダー人材の戦略的育成についてのガイドライン」においても、財務部門のリーダーには事業戦略の立案能力、データに基づく意思決定能力、そしてリスク管理能力が求められるとされている。

具体的には、以下のような場面でCFOの戦略的判断が不可欠となる。

- 新規事業への投資判断：NPV（正味現在価値）やIRR（内部収益率）だけでなく、戦略的適合性やシナジー効果を含めた総合的な評価が必要である。
- M&Aにおけるバリュエーション：買収対象企業の財務分析に加え、統合後のコストシナジーや売上シナジーを定量化する能力が求められる。
- 事業ポートフォリオの最適化：各事業の収益性をROIC（投下資本利益率）で評価し、資本コストを上回る事業に経営資源を集中させる判断を行う。

### 1-3.「過去の記録」から「未来の設計」へ

戦略会計の本質は、会計情報を「過去の記録」から「未来を設計するためのツール」へと転換することにある。伝統的な会計が「何が起きたか」を記録するのに対し、戦略会計は「何をすべきか」を明らかにする。

例えば、ある製品の売上が前年比10%減少したという事実を記録するだけでは、経営判断の材料としては不十分である。戦略会計の視点では、以下のような問いを立てる。

- その売上減少は市場全体の縮小によるものか、それとも競合へのシェア流出か。
- 競合製品との価格差はどの程度か、そしてその価格差を生み出しているコスト構造の違いは何か。
- 顧客の購買行動はどのように変化しているか、そして将来の需要はどう推移するか。

このような問いに答えるために、本書ではこれから12の章にわたって、戦略会計の主要フレームワークを体系的に解説していく。原価企画からAI活用に至るまで、各手法の理論的背景と実務への応用を、具体的な数値例を交えて詳述する。

## 第2章 原価企画（Target Costing）

### 2-1. トヨタが生んだ「逆算」の発想

原価企画（Target Costing）は、日本の自動車産業、とりわけトヨタ自動車が1960年代から体系化してきた原価管理手法である。欧米の伝統的な原価計算が「製造原価を積み上げて販売価格を決定する（コストプラス方式）」のに対し、原価企画は「市場が受け入れる価格から逆算して許容原価を決定する」という、まったく逆のアプローチをとる。

この逆算の発想が生まれた背景には、日本の自動車市場の厳しい競争環境がある。消費者が受け入れる価格帯は市場によって決まっており、メーカーが一方的にコストを上乗せした価格を設定しても、顧客は競合製品に流れてしまう。したがって、市場価格を所与として、そこから目標利益を確保できるだけのコスト水準を実現しなければならない。

### 2-2. 原価企画のプロセス

原価企画は、以下の4つのステップで進められる。

**ステップ1：目標販売価格の設定**

市場調査や競合分析に基づき、顧客が受け入れる販売価格を設定する。例えば、新型SUVの開発において、競合車種の価格帯が350万円〜420万円であり、自社のブランドポジションと搭載する機能を考慮して、目標販売価格を380万円と設定する。

**ステップ2：目標利益の決定**

中期経営計画で求められる利益率に基づき、目標利益を算出する。売上高営業利益率の目標が8%であれば、以下のとおりとなる。

    目標利益 = 380万円 x 8% = 30.4万円

**ステップ3：許容原価の算出**

目標販売価格から目標利益を差し引いて、許容原価を求める。

    許容原価 = 380万円 - 30.4万円 = 349.6万円

**ステップ4：成行原価との差額（原価低減目標）の把握**

現在の技術水準と生産方式で製造した場合に見込まれる原価（成行原価）を試算し、許容原価との差額を原価低減の目標とする。

    成行原価（試算） = 375万円
    原価低減目標 = 375万円 - 349.6万円 = 25.4万円

この25.4万円をいかに削減するかが、原価企画活動の核心となる。

### 2-3. VE活動とクロスファンクショナルチーム

原価低減目標を達成するために、トヨタをはじめとする日本企業ではVE（Value Engineering：価値工学）活動が広く用いられている。VEの基本思想は、製品の「機能（Function）」と「コスト（Cost）」の関係を最適化することである。

    価値（Value） = 機能（Function） / コスト（Cost）

価値を高めるには、(1)同じ機能をより低いコストで実現する、(2)同じコストでより高い機能を実現する、(3)コストを増やす以上に機能を高める、という3つの方法がある。

原価企画では、設計・開発段階から製造、購買、品質管理、マーケティングなど、各部門の担当者がクロスファンクショナルチームを編成し、VE活動に取り組む。設計段階で原価の80%以上が決定されるといわれており、量産後に原価を下げることは極めて困難である。だからこそ、設計段階での原価作り込みが重要となる。

### 2-4. 数値例：自動車部品の原価企画

あるブレーキシステムの原価企画を例に、具体的な数値で見てみよう。

    目標販売価格（自動車メーカーへの納入価格）：48,000円
    目標利益率：6%
    目標利益：48,000円 x 6% = 2,880円
    許容原価：48,000円 - 2,880円 = 45,120円
    成行原価：52,000円
    原価低減目標：52,000円 - 45,120円 = 6,880円

原価低減の内訳を、部品別・活動別に分解すると以下のようになる。

| 項目 | 成行原価 | 低減目標 | 目標原価 |
|------|---------|---------|---------|
| 材料費 | 28,000円 | -3,200円 | 24,800円 |
| 加工費 | 14,000円 | -2,000円 | 12,000円 |
| 組立費 | 6,000円 | -1,080円 | 4,920円 |
| 検査費 | 4,000円 | -600円 | 3,400円 |
| 合計 | 52,000円 | -6,880円 | 45,120円 |

材料費の低減は、代替材料の採用や部品点数の削減で実現する。加工費は工程の簡素化や自動化により、組立費は部品の統合（モジュール化）によって削減を図る。このように、原価企画は単なるコスト削減ではなく、製品設計そのものを見直す活動である。

## 第3章 活動基準原価計算（ABC: Activity-Based Costing）

### 3-1. 伝統的配賦の問題点

伝統的な原価計算では、製造間接費を直接作業時間や機械稼働時間などの単一の配賦基準で製品に配分する。しかし、製造業の自動化が進み、間接費の割合が増大するにつれて、この方法では製品原価を正確に把握できなくなった。

例えば、ある工場で製品Aと製品Bを製造しているとする。

    製品A：大量生産品、月産10,000個、シンプルな構造
    製品B：少量生産品、月産500個、複雑な構造

製造間接費の総額が月間2,000万円であり、これを直接作業時間（製品A：5,000時間、製品B：500時間）で配賦すると、以下のようになる。

    配賦率 = 2,000万円 / 5,500時間 = 3,636円/時間
    製品Aへの配賦額 = 3,636円 x 5,000時間 = 1,818万円（1個あたり1,818円）
    製品Bへの配賦額 = 3,636円 x 500時間 = 182万円（1個あたり3,636円）

しかし実際には、製品Bは少量多品種であるため、段取り替え、品質検査、個別の受注処理、特殊な材料の調達など、製品Aに比べてはるかに多くの間接的な活動を必要としている。直接作業時間による配賦では、製品Aに過大な間接費が配賦され、製品Bの間接費が過小に計算されてしまう。この「クロスサブシディ（内部補助）」の問題が、誤った価格設定や製品ミックスの判断につながる危険性がある。

### 3-2. コストドライバーと活動プール

ABC（Activity-Based Costing：活動基準原価計算）は、1980年代にハーバード・ビジネス・スクールのロバート・キャプラン教授らによって体系化された手法である。ABCでは、間接費を「活動（Activity）」単位で集計し、各活動の「コストドライバー（原価作用因）」に基づいて製品に配賦する。

ABCの基本的な考え方は以下のとおりである。

    製品が活動を消費する → 活動が資源（コスト）を消費する

主要なコストドライバーの例を挙げる。

- 段取り替え活動 → コストドライバー：段取り回数
- 品質検査活動 → コストドライバー：検査回数
- 受注処理活動 → コストドライバー：受注件数
- 材料調達活動 → コストドライバー：発注回数
- 機械保全活動 → コストドライバー：機械稼働時間

### 3-3. ABCと伝統的手法の比較：具体例

前述の製品AとBについて、ABCを適用した場合の原価計算を示す。

間接費2,000万円の内訳を活動別に分解すると以下のとおりとなる。

| 活動 | コスト | コストドライバー | 製品A | 製品B |
|------|--------|----------------|-------|-------|
| 段取り替え | 400万円 | 段取り回数 | 10回 | 40回 |
| 品質検査 | 300万円 | 検査回数 | 100回 | 200回 |
| 受注処理 | 200万円 | 受注件数 | 12件 | 48件 |
| 材料調達 | 300万円 | 発注回数 | 20回 | 80回 |
| 機械運転 | 800万円 | 機械時間 | 4,000時間 | 400時間 |
| 合計 | 2,000万円 | - | - | - |

ABCによる配賦計算は以下のようになる。

**段取り替え費：**

    配賦率 = 400万円 / 50回 = 80,000円/回
    製品A = 80,000円 x 10回 = 80万円
    製品B = 80,000円 x 40回 = 320万円

**品質検査費：**

    配賦率 = 300万円 / 300回 = 10,000円/回
    製品A = 10,000円 x 100回 = 100万円
    製品B = 10,000円 x 200回 = 200万円

**受注処理費：**

    配賦率 = 200万円 / 60件 = 33,333円/件
    製品A = 33,333円 x 12件 = 40万円
    製品B = 33,333円 x 48件 = 160万円

**材料調達費：**

    配賦率 = 300万円 / 100回 = 30,000円/回
    製品A = 30,000円 x 20回 = 60万円
    製品B = 30,000円 x 80回 = 240万円

**機械運転費：**

    配賦率 = 800万円 / 4,400時間 = 1,818円/時間
    製品A = 1,818円 x 4,000時間 = 727万円
    製品B = 1,818円 x 400時間 = 73万円

| 方式 | 製品A合計 | 製品A単価 | 製品B合計 | 製品B単価 |
|------|----------|----------|----------|----------|
| 伝統的手法 | 1,818万円 | 1,818円 | 182万円 | 3,636円 |
| ABC | 1,007万円 | 1,007円 | 993万円 | 19,860円 |

ABCによって、少量多品種の製品Bが実際にはきわめて高い間接費を消費していることが明らかになった。製品Bの1個あたり間接費は、伝統的手法の3,636円に対して、ABCでは19,860円と約5.5倍の差が生じている。この情報は、価格設定の見直しや製品ポートフォリオの再検討において極めて重要な示唆を与える。

### 3-4. ABCが最も有効な場面

ABCの導入は全ての企業に適しているわけではない。ABCが特に有効なのは以下のような状況である。

- 間接費の割合が総原価の30%以上を占める場合
- 製品の多様性が高く、少量多品種生産を行っている場合
- 製品ごとに消費する活動の量やパターンが大きく異なる場合
- 価格競争が激しく、正確な製品原価情報が競争力に直結する場合

逆に、製品の種類が少なく、間接費の割合も低い場合には、ABCの導入コストに見合った便益が得られない可能性が高い。日本では、花王やオムロンなどがABCを導入し、製品別の収益性分析に活用している事例が知られている。

## 第4章 バランスト・スコアカード（BSC）

### 4-1. 4つの視点

バランスト・スコアカード（BSC: Balanced Scorecard）は、1992年にハーバード・ビジネス・スクールのロバート・S・キャプラン教授とコンサルタントのデビッド・P・ノートンによって提唱された戦略的業績評価システムである。BSCの最大の特徴は、財務指標だけでなく、非財務指標を含めた4つの視点からバランスよく業績を評価する点にある。

**（1）財務の視点（Financial Perspective）**

株主や投資家に対して、財務的にどのような成果を出すべきかを示す。代表的なKPIとしては、売上高成長率、営業利益率、ROIC（投下資本利益率）、フリーキャッシュフローなどがある。

**（2）顧客の視点（Customer Perspective）**

ターゲット顧客に対して、どのような価値を提供すべきかを明確にする。顧客満足度、顧客維持率（リテンション率）、市場シェア、ネットプロモータースコア（NPS）などが代表的なKPIである。

**（3）内部プロセスの視点（Internal Business Process Perspective）**

財務目標と顧客目標を達成するために、どの業務プロセスを卓越させるべきかを特定する。製造リードタイム、不良品率、新製品開発サイクルタイム、プロセス効率などが指標として用いられる。

**（4）学習と成長の視点（Learning and Growth Perspective）**

長期的な成長のために、組織と人材をどのように強化すべきかを定義する。従業員満足度、研修時間、特許取得件数、IT投資額、従業員一人あたり提案件数などが該当する。

### 4-2. 戦略マップ：因果関係の連鎖

BSCの真価は、4つの視点を個別に管理することではなく、それらを戦略マップによって因果関係の連鎖（ストラテジー・マップ）として体系化することにある。

例えば、ある製造業の戦略マップは以下のように描ける。

    【学習と成長】技術者のスキル向上 → 生産技術研修の充実
        ↓
    【内部プロセス】製造不良率の低減 → 生産リードタイムの短縮
        ↓
    【顧客】品質向上による顧客満足度の改善 → リピート率の向上
        ↓
    【財務】売上高の増加 → 営業利益率の改善 → ROIC目標の達成

この因果関係の連鎖が重要である理由は、現場の具体的な活動と最終的な財務成果がどのようにつながっているかを、組織全体で共有できるからである。技術者のスキル向上という日常の取り組みが、最終的にROICの改善につながるという論理が明確になることで、現場のモチベーション向上にも寄与する。

### 4-3. KPI設計の実践

各視点のKPIは、以下の原則に基づいて設計する。

- SMART原則：Specific（具体的）、Measurable（測定可能）、Achievable（達成可能）、Relevant（戦略と関連）、Time-bound（期限付き）であること。
- 先行指標と遅行指標のバランス：財務指標は遅行指標（結果指標）であり、顧客や内部プロセスの指標は先行指標（パフォーマンスドライバー）として機能する。両者をバランスよく配置する。
- KPIの数は20個以内に絞る：多すぎるKPIは経営者の注意を分散させる。各視点4〜5個程度が適切である。

具体的なKPI設計の例を示す。

| 視点 | 戦略目標 | KPI | 目標値 |
|------|---------|-----|--------|
| 財務 | 収益性の向上 | ROIC | 10%以上 |
| 財務 | 売上成長 | 売上高成長率 | 年率5% |
| 顧客 | 顧客ロイヤルティ強化 | NPS | 50以上 |
| 顧客 | 新規顧客獲得 | 新規顧客数 | 月100社 |
| 内部プロセス | 品質改善 | 不良品率 | 0.1%以下 |
| 内部プロセス | 開発速度向上 | 開発リードタイム | 6ヶ月以内 |
| 学習と成長 | 人材育成 | 一人あたり研修時間 | 年40時間 |
| 学習と成長 | イノベーション | 特許出願件数 | 年50件 |

### 4-4. 導入の課題と解決策

BSCの導入において、日本企業がしばしば直面する課題とその解決策を整理する。

**課題1：形骸化**
BSCを導入したものの、KPIの設定と報告が形式的な作業に陥り、実際の経営判断に活用されないケースがある。解決策として、経営会議でBSCのレビューを定例議題とし、KPIの達成状況に基づいてアクションプランを策定・実行するPDCAサイクルを確立することが重要である。

**課題2：因果関係の検証不足**
戦略マップで想定した因果関係が実際に成立しているかを検証しないまま運用されることがある。例えば「従業員満足度が上がれば顧客満足度が上がる」という仮説が本当に成立しているか、統計的な分析で定期的に確認する必要がある。

**課題3：部門間の整合性**
全社BSCと各部門BSCの整合性が取れていない場合、部門最適と全体最適が乖離する。カスケーディング（上位BSCの目標を下位組織の目標に展開する作業）を丁寧に行うことが不可欠である。

## 第5章 CVP分析（損益分岐点分析）

### 5-1. 固定費と変動費の分解

CVP分析（Cost-Volume-Profit Analysis）は、コスト・販売量・利益の関係を分析し、経営判断に役立てる手法である。日本語では「損益分岐点分析」と呼ばれることが多い。CVP分析の前提として、すべてのコストを変動費と固定費に分解する必要がある。

**変動費（Variable Cost）：** 販売量や生産量に比例して増減するコスト。材料費、販売手数料、外注加工費などが該当する。

**固定費（Fixed Cost）：** 販売量や生産量に関係なく一定額が発生するコスト。減価償却費、賃借料、正社員の給与（基本給部分）、保険料などが該当する。

実務上、すべてのコストを厳密に変動費か固定費に分類することは難しい。例えば、電力料金は基本料金（固定費的）と使用量に応じた従量料金（変動費的）の両方を含む準変動費である。このような場合、高低点法や回帰分析法を用いて、固定費部分と変動費部分を推定する。

**高低点法の例：**

ある工場の過去6ヶ月の生産量と電力費のデータが以下のとおりであるとする。

    最高月：生産量 12,000個、電力費 480万円
    最低月：生産量 6,000個、電力費 330万円

    変動費率 = (480万円 - 330万円) / (12,000個 - 6,000個) = 250円/個
    固定費 = 480万円 - 250円 x 12,000個 = 180万円

### 5-2. 損益分岐点の算出

損益分岐点（Break-Even Point: BEP）とは、売上高と総費用が等しくなり、利益がちょうどゼロになる販売量（または売上高）のことである。

    利益 = 売上高 - 変動費 - 固定費
    損益分岐点では利益 = 0 であるから、
    売上高 = 変動費 + 固定費

**数値例：**

    販売価格：2,000円/個
    変動費：1,200円/個
    固定費：月額4,000万円

    貢献利益（1個あたり） = 2,000円 - 1,200円 = 800円
    損益分岐点販売量 = 4,000万円 / 800円 = 50,000個
    損益分岐点売上高 = 50,000個 x 2,000円 = 1億円

これは貢献利益率を用いて次のようにも計算できる。

    貢献利益率 = 800円 / 2,000円 = 40%
    損益分岐点売上高 = 4,000万円 / 40% = 1億円

### 5-3. 貢献利益分析

貢献利益（Contribution Margin）は、売上高から変動費を差し引いた金額であり、固定費の回収と利益の創出に「貢献」する部分である。貢献利益の概念は、製品の継続・廃止の判断、受注の可否判断、セグメント別の収益性分析など、多くの経営判断に活用される。

**特別注文の受諾判断の例：**

通常販売価格2,000円の製品について、大口顧客から1,500円で10,000個の特別注文があったとする。変動費は1,200円/個であり、この注文を引き受けても固定費の増加はないとする。

    特別注文の貢献利益 = (1,500円 - 1,200円) x 10,000個 = 300万円

通常価格を下回る注文ではあるが、変動費を上回る価格であれば貢献利益が生じ、固定費の回収に追加的に貢献する。遊休生産能力がある場合、この注文を受諾することで利益は300万円改善する。ただし、既存顧客への価格への影響（値崩れリスク）を慎重に検討する必要がある。

### 5-4. 複数製品のCVP分析

現実の企業では複数の製品を販売しているため、製品ミックスを考慮したCVP分析が必要となる。

**数値例：**

| 製品 | 販売価格 | 変動費 | 貢献利益 | 販売構成比 |
|------|---------|--------|---------|-----------|
| X | 3,000円 | 1,800円 | 1,200円 | 40% |
| Y | 2,000円 | 1,000円 | 1,000円 | 35% |
| Z | 5,000円 | 3,500円 | 1,500円 | 25% |

    固定費：月額6,000万円

    加重平均貢献利益率を計算する。
    製品X：(1,200/3,000) x 40% = 16.0%
    製品Y：(1,000/2,000) x 35% = 17.5%
    製品Z：(1,500/5,000) x 25% = 7.5%
    加重平均貢献利益率 = 16.0% + 17.5% + 7.5% = 41.0%

    損益分岐点売上高 = 6,000万円 / 41.0% = 約14,634万円

### 5-5. 安全余裕率と営業レバレッジ

**安全余裕率（Margin of Safety）** は、実際の売上高が損益分岐点売上高をどれだけ上回っているかを示す指標である。

    安全余裕率 = (実際売上高 - 損益分岐点売上高) / 実際売上高 x 100%

例えば、実際売上高が2億円、損益分岐点売上高が1億円であれば、安全余裕率は50%である。売上高が50%減少しても赤字にならないことを意味し、経営の安定性を示す。一般的に、安全余裕率が20%以上であれば安全と考えられる。

**営業レバレッジ（Operating Leverage）** は、売上高の変化に対して営業利益がどれだけ敏感に反応するかを示す指標である。

    営業レバレッジ係数 = 貢献利益 / 営業利益

固定費の割合が高い企業ほど営業レバレッジが大きく、売上増加時には利益が大きく伸びるが、売上減少時には利益が急速に縮小するリスクがある。例えば、営業レバレッジが3倍の企業では、売上が10%増加すると営業利益は30%増加するが、売上が10%減少すると営業利益は30%減少する。

## 第6章 バリューチェーン分析

### 6-1. ポーターのバリューチェーン・フレームワーク

バリューチェーン分析は、マイケル・ポーター教授が1985年の著書『競争優位の戦略』で提唱したフレームワークである。企業の事業活動を「主活動」と「支援活動」に分類し、各活動がどのように価値を創造し、コストを発生させているかを体系的に分析する。

**主活動（Primary Activities）：**

1. 購買物流（Inbound Logistics）：原材料の受入、保管、在庫管理
2. 製造（Operations）：原材料を最終製品に変換する活動
3. 出荷物流（Outbound Logistics）：完成品の保管、配送、流通
4. マーケティング・販売（Marketing & Sales）：広告、販促、価格設定、チャネル管理
5. サービス（Service）：アフターサービス、修理、顧客サポート

**支援活動（Support Activities）：**

1. 企業インフラ（Firm Infrastructure）：経営管理、財務、法務、経理
2. 人事・労務管理（Human Resource Management）：採用、研修、報酬管理
3. 技術開発（Technology Development）：R&D、製品設計、プロセス改善
4. 調達活動（Procurement）：原材料や設備の購買活動

### 6-2. バリューチェーン全体のコスト分析

戦略的コスト管理の要諦は、自社のバリューチェーンだけでなく、サプライヤーから最終顧客に至る業界全体のバリューチェーンを俯瞰してコスト構造を理解することにある。

例えば、ある食品メーカーのバリューチェーン全体のコスト構造を分析すると以下のようになる。

| 活動 | コスト構成比 | 主なコスト項目 |
|------|------------|--------------|
| 原材料調達 | 35% | 農産物仕入、輸送費 |
| 製造 | 20% | 人件費、設備減価償却、光熱費 |
| 品質管理 | 5% | 検査費、認証取得費 |
| 包装 | 8% | 包装材料、デザイン費 |
| 物流・配送 | 12% | 倉庫費、配送費、冷蔵費 |
| マーケティング | 10% | 広告費、販促費 |
| 販売・営業 | 7% | 営業人件費、販売手数料 |
| アフターサービス | 3% | 問い合わせ対応、返品処理 |

この分析から、原材料調達（35%）が最大のコスト要因であることがわかる。したがって、コスト競争力を高めるには、調達先の多様化、長期契約による価格安定化、あるいは原材料の内製化（垂直統合）が有効な戦略となりうる。

### 6-3. 競合他社との比較分析

バリューチェーン分析の戦略的な価値は、自社と競合他社のコスト構造を比較することで明確になる。同業他社が自社よりも低いコストで特定の活動を遂行している場合、その差異の原因を特定し、改善策を講じる必要がある。

例えば、日本のコンビニエンスストア業界において、セブン-イレブンが高い収益性を維持している要因の一つは、物流システムの効率性にある。温度帯別の共同配送システムにより、1店舗あたりの配送回数を削減しつつ、鮮度の高い商品供給を実現している。これは、バリューチェーンにおける「出荷物流」の活動で競争優位を確立した好例である。

### 6-4. 戦略的コスト管理：投資と削減の判断

バリューチェーン分析に基づく戦略的コスト管理では、すべての活動のコストを一律に削減するのではなく、競争優位の源泉となる活動にはむしろ投資を行い、差別化に寄与しない活動のコストを削減するという、メリハリのある判断が重要である。

トヨタ自動車を例にとると、製造工程におけるTPS（トヨタ生産方式）の改善活動や、サプライヤーとの共同開発活動には継続的に投資する一方、間接部門の業務効率化やITシステムの共通化によるコスト削減を同時に推進している。

戦略的コスト管理の判断基準として、以下の2軸で各活動を評価するフレームワークが有効である。

- 軸1：その活動は顧客価値の創造に直結しているか
- 軸2：その活動は競合他社との差別化に寄与しているか

両方にYesと答えられる活動は「コア活動」として投資を強化する。いずれもNoの活動は、アウトソーシングや自動化によるコスト最小化の対象とする。

## 第7章 ライフサイクル・コスティング

### 7-1. 製品ライフサイクルとコスト行動

ライフサイクル・コスティング（Life Cycle Costing: LCC）は、製品の企画・開発段階から製造、販売、使用、廃棄に至るまでの全ライフサイクルにわたるコストを把握・管理する手法である。伝統的な原価計算が製造段階のコストに焦点を当てるのに対し、LCCは「ゆりかごから墓場まで」の総コストを対象とする。

製品ライフサイクルは、一般に以下の段階で構成される。

**（1）企画・開発段階：** 市場調査、コンセプト開発、設計、試作、テスト
**（2）導入段階：** 生産立ち上げ、初期マーケティング、流通チャネル構築
**（3）成長段階：** 生産量拡大、市場浸透、競合対応
**（4）成熟段階：** 生産効率最適化、製品改良、コスト削減
**（5）衰退段階：** 生産縮小、在庫処分、製品撤退

各段階のコスト構造は大きく異なる。企画・開発段階では研究開発費や設計費が中心であり、製造段階では材料費や加工費が、衰退段階では在庫処分費や設備撤去費が主要なコストとなる。

### 7-2. 上流コストと下流コスト

LCCの重要な知見は、製品コストの大部分が上流段階（企画・設計）で決定されるという事実である。研究によれば、製品の総ライフサイクルコストの80〜85%は設計段階で確定すると言われている。

    総ライフサイクルコストの構成例（自動車部品）：
    企画・設計段階で確定するコスト：82%
    製造段階で変更可能なコスト：12%
    販売・アフターサービス段階で管理可能なコスト：6%

一方、実際にコストが発生（支出）するのは製造段階以降である。つまり、コストの大半は設計段階で「決定」されるが、実際に「支出」されるのは後工程である。このギャップを認識することが、LCCの戦略的意義である。

上流段階での意思決定がいかに重要かを示す具体例として、自動車の車体構造の設計を考える。スチール製の車体をアルミニウム製に変更する場合、設計段階では材料選定と構造設計の見直しが必要となる。アルミニウムの材料費はスチールの約3倍であるが、軽量化による燃費改善は製品の全使用期間にわたって顧客に価値を提供し、環境規制への対応コストを低減させる。このようなトレードオフの判断は、ライフサイクル全体を見渡さなければ適切に行えない。

### 7-3. 総所有コスト（TCO）の視点

LCCの考え方を購買者側に適用したのが、TCO（Total Cost of Ownership：総所有コスト）である。TCOは、購入価格だけでなく、運用コスト、保守コスト、廃棄コストを含めた総コストで購買判断を行う考え方である。

**産業用機械のTCO比較例：**

| 項目 | A社製品 | B社製品 |
|------|--------|--------|
| 購入価格 | 5,000万円 | 6,500万円 |
| 年間保守費 | 300万円 | 150万円 |
| 年間電力費 | 200万円 | 120万円 |
| 10年間修理費合計 | 800万円 | 300万円 |
| 廃棄・撤去費 | 200万円 | 100万円 |
| 10年間TCO | 10,000万円 | 9,600万円 |

購入価格ではA社製品が1,500万円安いが、10年間のTCOではB社製品の方が400万円安い。TCOの視点で見ることで、長期的に合理的な購買判断が可能となる。

### 7-4. ライフサイクル段階別の戦略的価格設定

ライフサイクルの各段階では、異なる価格戦略が求められる。

**導入段階：** 浸透価格戦略（低価格で市場シェアを迅速に獲得）または上澄み吸収価格戦略（高価格で初期投資を早期回収）を選択する。ソニーの新型ゲーム機は上澄み吸収戦略、多くのSaaS企業は浸透価格戦略を採用している。

**成長段階：** 競合の参入に応じて価格を調整しつつ、規模の経済によるコスト低減を価格に反映させる。

**成熟段階：** 製品差別化や付加サービスにより価格を維持するか、コストリーダーシップにより低価格を実現する。

**衰退段階：** 限界的な顧客への価格引き下げ、あるいは後継製品への移行促進のための価格政策を実施する。在庫処分コストと値引き幅のバランスが判断のポイントとなる。

## 第8章 制約理論（TOC: Theory of Constraints）

### 8-1. ゴールドラットの5段階集中プロセス

制約理論（TOC: Theory of Constraints）は、イスラエルの物理学者エリヤフ・ゴールドラットが1984年の著書『ザ・ゴール』で提唱した経営理論である。TOCの基本思想は、あらゆるシステムには少なくとも1つの制約（ボトルネック）が存在し、システム全体のパフォーマンスはその制約によって決定されるというものである。

TOCの改善プロセスは、以下の5段階で構成される。

**ステップ1：制約を特定する（Identify）**

システム全体のスループットを制限しているボトルネック工程を特定する。例えば、生産ライン上で最も処理能力が低い工程、最も待ち時間が長い工程がボトルネックである。

**ステップ2：制約を最大限に活用する（Exploit）**

ボトルネック工程の稼働率を最大化する。休憩時間の分散、段取り替え時間の短縮、不良品のボトルネック前工程での排除などにより、ボトルネックの有効稼働時間を増やす。

**ステップ3：他のすべてを制約に従属させる（Subordinate）**

非ボトルネック工程の生産ペースをボトルネック工程に合わせる。非ボトルネック工程がボトルネック工程よりも速く生産しても、中間在庫が増えるだけでシステム全体のスループットは改善しない。

**ステップ4：制約を強化する（Elevate）**

設備の増設、人員の追加、技術革新などにより、ボトルネック工程の処理能力を根本的に向上させる。このステップには投資が伴うため、費用対効果を慎重に検討する。

**ステップ5：慣性に注意して最初に戻る（Repeat）**

ステップ4でボトルネックが解消されると、別の工程が新たなボトルネックになる可能性がある。常に最新の制約を特定し、改善サイクルを継続する。

### 8-2. スループット会計と伝統的原価計算の違い

TOCに基づくスループット会計では、以下の3つの指標で業績を評価する。

**スループット（T）：** 売上高から真に変動する費用（材料費など）を差し引いた金額。

    T = 売上高 - 真の変動費

**在庫/投資（I）：** システムが販売するために投資したすべての資金。原材料、仕掛品、完成品在庫、設備投資額を含む。

**業務費用（OE）：** スループットを生み出すためにシステムが費やすすべての費用。人件費、間接費、減価償却費などが該当する。

    純利益 = T - OE
    投資利益率 = (T - OE) / I

伝統的原価計算との最大の違いは、TOCでは労務費を変動費ではなく業務費用（固定費的）として扱う点である。これにより、「生産を増やすほど1個あたりの原価が下がる」という伝統的な発想から脱却し、在庫を積み上げることが利益をもたらすという錯覚を排除できる。

### 8-3. ボトルネック管理の数値例

ある工場に工程A、B、Cの3工程があり、以下のデータが与えられている。

    工程A：処理能力 100個/時間
    工程B：処理能力 60個/時間（ボトルネック）
    工程C：処理能力 80個/時間

    製品販売価格：5,000円/個
    材料費：1,500円/個
    スループット：3,500円/個

この場合、システム全体の生産能力はボトルネックである工程Bの60個/時間に制限される。

    1日（8時間稼働）のスループット = 3,500円 x 60個 x 8時間 = 168万円

ここで、ボトルネック1分あたりのスループットを計算する。

    ボトルネック1分あたりのスループット = 3,500円 x 60個 / 60分 = 3,500円/分

この指標は、複数の製品がボトルネック工程を共有している場合の製品ミックス判断に極めて有用である。ボトルネック1分あたりのスループットが高い製品を優先的に生産することで、システム全体のスループットを最大化できる。

### 8-4. DBR（ドラム・バッファ・ロープ）スケジューリング

DBRはTOCに基づく生産スケジューリング手法であり、3つの要素で構成される。

**ドラム（Drum）：** ボトルネック工程の生産スケジュール。ドラムの鼓動のように、全体の生産ペースを決定する。

**バッファ（Buffer）：** ボトルネック工程の前に設ける時間的余裕。上流工程のトラブルによってボトルネックが停止することを防ぐための安全在庫である。

**ロープ（Rope）：** 最初の工程への材料投入をボトルネックの消費ペースに連動させるメカニズム。ロープで投入量を制御することで、仕掛品在庫の過剰な蓄積を防ぐ。

DBRの運用により、仕掛品在庫を最小化しつつ、ボトルネック工程の稼働率を最大化するという、一見矛盾する二つの目標を同時に達成できる。

## 第9章 アメーバ経営と時間当たり採算

### 9-1. 稲盛和夫の経営哲学

アメーバ経営は、京セラの創業者である稲盛和夫氏が構築した独自の経営管理手法である。その根底にあるのは「全員参加の経営」という理念であり、経営を一部の経営幹部に任せるのではなく、すべての従業員が経営者意識を持ってそれぞれの持ち場で最善を尽くすという思想に基づいている。

稲盛氏は京セラの経営において、次の3つの課題に直面した。

1. 急成長する組織において、経営の実態をリアルタイムに把握すること
2. すべての従業員に経営者意識を浸透させること
3. 大企業病を防ぎ、小さな組織の機動力を維持すること

これらの課題を同時に解決するために生み出されたのが、組織を小さな独立採算単位（アメーバ）に分割し、各アメーバの収支をリアルタイムに管理するアメーバ経営である。

京セラは1959年の創業以来、一度も赤字決算を出すことなく成長を続けてきた。また、2010年に稲盛氏が会長として日本航空（JAL）の再建を指揮した際にも、アメーバ経営の手法を導入し、わずか2年で営業利益率10%以上の高収益企業への転換を実現した。

### 9-2. アメーバ（小集団独立採算）の仕組み

アメーバ経営では、企業を5〜50名程度の小さな組織単位（アメーバ）に分割する。各アメーバにはリーダーが任命され、そのリーダーは自部門の売上とコストに対して経営責任を負う。

アメーバの編成にあたっては、以下の3つの条件を満たす必要がある。

**条件1：独立採算が可能であること。** 各アメーバが自らの売上と経費を明確に把握できる組織単位であること。

**条件2：事業として完結していること。** 各アメーバが独立した事業としての実体を持ち、リーダーが自部門の経営を計画し実行できること。

**条件3：会社全体の目的と整合していること。** 個々のアメーバの最適化が全社の最適化と矛盾しないよう、組織の分割方法を工夫すること。

アメーバ間の取引には「社内売買」の仕組みが適用される。例えば、製造アメーバが営業アメーバに製品を「販売」し、営業アメーバがそれを外部顧客に販売する。社内売買価格は、市場価格に基づいて設定されるか、あるいは上位組織の判断によって決定される。

### 9-3. 時間当たり採算の計算方法

アメーバ経営の核心的な管理指標が「時間当たり採算」である。これは、各アメーバの付加価値を総労働時間で割って算出される。

    時間当たり採算 = (総収入 - 経費) / 総労働時間

ここで注意すべきは、「経費」には人件費を含まないという点である。これは一般的な管理会計の発想とは異なるが、稲盛氏の哲学には明確な理由がある。人件費を経費に含めると、人員を削減するほど時間当たり採算が改善するように見えるが、それでは「全員参加の経営」という理念に反する。人件費を除外することで、各アメーバは人を減らすのではなく、同じメンバーでより多くの付加価値を生み出すことに集中する。

**時間当たり採算の計算例：**

ある製造アメーバの月次データが以下のとおりであるとする。

    社内売上（営業アメーバへの売上）：1,200万円
    社外からの受注売上：300万円
    材料費：500万円
    外注費：150万円
    その他経費（光熱費、消耗品等）：100万円
    メンバー数：10名
    総労働時間：1,800時間（10名 x 180時間/月）

    差引収益 = (1,200万円 + 300万円) - (500万円 + 150万円 + 100万円) = 750万円
    時間当たり採算 = 750万円 / 1,800時間 = 4,167円/時間

この「4,167円/時間」という数字が、そのアメーバの生産性を端的に表す。月次でこの数字を追跡し、前月比や年間目標との比較を行うことで、改善活動のPDCAサイクルを回す。

### 9-4. 従業員のエンゲージメント向上

アメーバ経営が従業員のエンゲージメントを高める仕組みは、以下のメカニズムによるものである。

**見える化：** 時間当たり採算という単一の指標に経営状態を集約することで、財務知識のない現場の従業員でも自部門の業績を直感的に理解できる。

**当事者意識：** 自らの行動が時間当たり採算に直結するため、コスト意識と売上意識が自然に芽生える。例えば、残業して生産量を増やしても、総労働時間が増加するため、時間当たり採算は期待ほど改善しない。むしろ、定時内での生産性向上が求められる。

**日次管理：** 京セラでは、アメーバの業績を日次で集計・報告する仕組みを構築している。月末にならないと業績がわからないのではなく、昨日の結果が翌朝にはわかるというリアルタイム性が、迅速な改善行動を促す。

**公正な評価：** 時間当たり採算は、大きなアメーバも小さなアメーバも同じ基準で比較可能であり、組織規模の違いによるバイアスが少ない。

## 第10章 品質原価計算（Quality Costing）

### 10-1. PAFモデル：予防・評価・失敗コスト

品質原価計算（Quality Costing）は、品質に関連するすべてのコストを体系的に把握し、品質改善活動の経済的効果を可視化するための手法である。最も広く用いられているフレームワークが、PAF（Prevention-Appraisal-Failure）モデルである。

**（1）予防コスト（Prevention Cost）**

品質問題が発生することを未然に防ぐために投じるコストである。具体的には以下の項目が該当する。

- 品質計画の策定費用
- 従業員への品質教育・研修費
- 工程能力調査費
- 設計レビュー費用
- サプライヤーの品質監査費
- 予防保全（PM）の実施費用

**（2）評価コスト（Appraisal Cost）**

製品やサービスが品質基準を満たしているかを確認するためのコストである。

- 受入検査費用
- 工程内検査費用
- 最終検査費用
- 検査機器の校正・維持費
- 外部認証（ISO等）の取得・維持費

**（3）内部失敗コスト（Internal Failure Cost）**

出荷前に発見された品質不良に対処するコストである。

- 手直し・再加工費
- スクラップ（廃棄）損失
- 再検査費用
- 不良原因の調査費
- 歩留まり低下による損失

**（4）外部失敗コスト（External Failure Cost）**

出荷後に顧客の手元で発見された品質不良に対処するコストである。

- クレーム対応費
- 製品保証（warranty）費用
- リコール費用
- 製品責任（PL）訴訟費用
- 信用失墜による売上減少（機会損失）

### 10-2. 品質コストの測定

品質コストを定量的に測定する際、多くの企業が直面するのは、品質関連コストが通常の会計システムでは捕捉されていないという問題である。例えば、「手直し作業に費やした時間」は労務費に含まれており、品質コストとして別途集計されていない。

品質コストの測定例（年商100億円の製造業）を示す。

| 品質コスト分類 | 金額 | 売上比 |
|--------------|------|--------|
| 予防コスト | 4,000万円 | 0.4% |
| 評価コスト | 8,000万円 | 0.8% |
| 内部失敗コスト | 12,000万円 | 1.2% |
| 外部失敗コスト | 6,000万円 | 0.6% |
| 品質コスト合計 | 30,000万円 | 3.0% |

一般的に、品質コストは売上高の5〜25%に達するとされている。上記の例では3.0%であり、品質管理が比較的良好であることを示しているが、それでも3億円のコストが品質に関連して発生している。

### 10-3. 最適品質コストバランス

品質コストの構成には重要なトレードオフの関係がある。予防コストと評価コスト（適合コスト）を増やすと、失敗コスト（不適合コスト）は減少する。逆に、予防・評価への投資を削ると、失敗コストが増大する。

伝統的な品質コストモデルでは、これらのトレードオフによって「最適品質コスト」が存在するとされた。すなわち、適合コストと不適合コストの合計が最小となる品質水準が経済的最適点である。

しかし、現代の品質管理思想（TQMやシックスシグマ）では、予防コストへの投資を増やすことで失敗コストを劇的に削減でき、品質コスト全体も低下するという考え方が主流である。つまり、品質向上と総コスト削減は両立しうる。

実際に、トヨタ自動車やデンソーなどの企業では、予防活動への積極的な投資によって不良率を極限まで低減し、手直し・リコール等の失敗コストを大幅に削減した結果、品質コストの総額を売上高の1%未満に抑えている事例がある。

### 10-4. シックスシグマと品質コスト削減

シックスシグマ（Six Sigma）は、モトローラが1980年代に開発し、GE（ゼネラル・エレクトリック）のジャック・ウェルチ会長のもとで広く普及した品質改善手法である。統計的手法を用いて、プロセスのばらつきを100万機会あたり3.4回の不良（シックスシグマ水準）まで低減することを目指す。

シグマ水準と品質コストの関係は以下のとおりである。

| シグマ水準 | 不良率（DPMO） | 品質コスト（売上比） |
|-----------|--------------|-------------------|
| 2シグマ | 308,537 | 30〜40% |
| 3シグマ | 66,807 | 20〜30% |
| 4シグマ | 6,210 | 15〜20% |
| 5シグマ | 233 | 5〜15% |
| 6シグマ | 3.4 | 1%未満 |

DPMO = Defects Per Million Opportunities（100万機会あたりの不良数）

日本企業では、シックスシグマを明示的に標榜する企業は多くないが、TQM活動やQCサークルを通じて、実質的にシックスシグマに匹敵する品質水準を達成している企業は少なくない。

## 第11章 ROIC経営とWACC

### 11-1. ROIC（投下資本利益率）の分解

ROIC（Return on Invested Capital：投下資本利益率）は、企業が投下した資本に対してどれだけの利益を生み出しているかを測定する指標であり、近年、日本企業において急速に普及している経営指標である。

    ROIC = NOPAT / 投下資本
    NOPAT = 営業利益 x (1 - 実効税率)
    投下資本 = 有利子負債 + 株主資本（= 事業資産 - 事業負債）

ROICが注目される理由は、ROE（自己資本利益率）やROA（総資産利益率）と比較して、事業の本質的な収益力をより正確に反映するからである。ROEは財務レバレッジ（借入）によって操作可能であり、ROAには事業と直接関係のない資産（余剰現金、投資有価証券等）が含まれる。ROICはこれらの問題を排除し、事業に投下された資本がどれだけ効率的に利益を生んでいるかを純粋に測定する。

### 11-2. WACC（加重平均資本コスト）の計算

WACC（Weighted Average Cost of Capital：加重平均資本コスト）は、企業が調達した資本の平均的なコスト（投資家が要求するリターン）を表す。

    WACC = D/(D+E) x rD x (1-t) + E/(D+E) x rE

    D：有利子負債の時価
    E：株主資本の時価（時価総額）
    rD：負債コスト（借入金利）
    rE：株主資本コスト
    t：実効税率

**計算例：**

    有利子負債：300億円、借入金利 2.0%
    時価総額：700億円、株主資本コスト 8.0%
    実効税率：30%

    WACC = 300/(300+700) x 2.0% x (1-30%) + 700/(300+700) x 8.0%
         = 30% x 1.4% + 70% x 8.0%
         = 0.42% + 5.60%
         = 6.02%

株主資本コスト（rE）は直接観察できないため、通常はCAPM（資本資産評価モデル）を用いて推定する。

    rE = Rf + beta x (Rm - Rf)
    Rf：リスクフリーレート（10年国債利回り等）
    beta：個別企業のベータ値
    Rm - Rf：マーケットリスクプレミアム

例えば、リスクフリーレートが1.0%、ベータが1.2、マーケットリスクプレミアムが6.0%であれば、

    rE = 1.0% + 1.2 x 6.0% = 8.2%

### 11-3. EVA（経済的付加価値）

EVA（Economic Value Added：経済的付加価値）は、スターン・スチュワート社が商標登録した指標であり、企業が資本コストを上回る利益を生み出しているかを測定する。

    EVA = NOPAT - 投下資本 x WACC
        = (ROIC - WACC) x 投下資本

EVAがプラスであれば、企業は資本コストを上回るリターンを創出しており、企業価値が増大している。EVAがマイナスであれば、資本コストを賄えておらず、企業価値を毀損している。

**計算例：**

    NOPAT：70億円
    投下資本：1,000億円
    WACC：6.02%

    EVA = 70億円 - 1,000億円 x 6.02% = 70億円 - 60.2億円 = 9.8億円

この企業はEVAが9.8億円のプラスであり、資本コストを超過する価値を創出している。ROICは7.0%（= 70億円 / 1,000億円）であり、WACC 6.02%を上回っている。この差（ROIC - WACC = 0.98%）を「エコノミック・スプレッド」と呼び、スプレッドがプラスであることが価値創造の条件である。

### 11-4. ROICツリーによるオペレーション改善

ROIC経営の実務上の強みは、ROICを構成要素に分解する「ROICツリー」を用いて、現場レベルの改善活動と全社的な資本効率の改善を結びつけられる点にある。

    ROIC = NOPAT / 投下資本
         = (NOPAT/売上高) x (売上高/投下資本)
         = NOPATマージン x 投下資本回転率

さらに、NOPATマージンと投下資本回転率をそれぞれ分解する。

**NOPATマージンの分解：**

    NOPATマージン = 売上総利益率 - 販管費率 - 税効果
    売上総利益率 → 材料費率、労務費率、製造間接費率
    販管費率 → 人件費率、広告宣伝費率、物流費率、研究開発費率

**投下資本回転率の分解：**

    投下資本回転率 = 売上高 / (運転資本 + 固定資産)
    運転資本 = 売上債権 + 棚卸資産 - 仕入債務
    → 売上債権回転日数、棚卸資産回転日数、仕入債務回転日数
    固定資産 → 有形固定資産回転率、無形固定資産回転率

オムロンやリコーなど日本の先進企業では、このROICツリーを事業部別に展開し、各事業部のリーダーが自部門のROIC改善に責任を持つ体制を構築している。例えば、オムロンは2014年からROIC経営を本格導入し、全事業部にROIC目標を設定するとともに、ROICツリーに基づく改善活動を推進している。これにより、事業ポートフォリオの入れ替えと各事業の収益力向上を同時に進め、企業価値の持続的な向上を実現してきた。

## 第12章 AIと戦略会計の未来

### 12-1. AIによるコスト分析と予測

人工知能（AI）の発展は、戦略会計の実務を根本的に変革しつつある。従来、管理会計担当者が手作業で行っていたデータ収集、分類、分析の多くが、AIによって自動化・高度化されている。

**コスト分析への適用：**

AIを活用することで、膨大な取引データから異常値やパターンを自動的に検出できるようになった。例えば、過去5年間の調達データを機械学習モデルに学習させることで、特定のサプライヤーからの購入価格が市場平均と乖離し始めた場合に自動的にアラートを発するシステムが実現している。

従来のABC（活動基準原価計算）においても、AIの活用により精度と効率が大幅に向上している。コストドライバーの特定にはこれまで管理会計の専門家の経験と判断が必要であったが、機械学習アルゴリズムを用いることで、データから最も説明力の高いコストドライバーを自動的に発見できる。

**予測分析への適用：**

時系列分析と機械学習を組み合わせた予測モデルにより、原材料価格の変動予測、為替レートの影響シミュレーション、需要予測に基づく最適在庫水準の算出などが可能になっている。例えば、ある大手製造業では、AIによる需要予測の精度を従来の統計モデル比で15〜20%改善し、在庫回転率を1.3倍に向上させた事例が報告されている。

### 12-2. 機械学習による需要予測

需要予測は、戦略会計における重要な入力情報である。CVP分析、原価企画、ライフサイクル・コスティングのいずれも、正確な需要予測を前提としている。

従来の需要予測手法（移動平均法、指数平滑法、ARIMA等）は、時系列データの過去パターンの延長線上で将来を予測する。これに対し、機械学習ベースの予測モデルは、時系列データに加えて、天候データ、SNSでの言及件数、マクロ経済指標、競合の価格変動、プロモーション計画など、多様な外部データを統合的に分析できる。

**機械学習予測の精度向上例：**

| 手法 | MAPE（平均絶対パーセント誤差） |
|------|---------------------------|
| 移動平均法 | 25〜30% |
| 指数平滑法 | 18〜25% |
| ARIMAモデル | 15〜20% |
| 勾配ブースティング | 10〜15% |
| ディープラーニング（LSTM） | 8〜12% |
| アンサンブルモデル | 7〜10% |

MAPEが10%改善することの経営的インパクトは極めて大きい。予測精度の向上は、過剰在庫の削減（保管コスト・廃棄ロスの低減）と機会損失の回避（欠品による販売逸失の防止）の両方に貢献する。

### 12-3. リアルタイム管理会計ダッシュボード

クラウドコンピューティングとBI（Business Intelligence）ツールの普及により、管理会計情報のリアルタイム可視化が実現している。従来は月次決算を待たなければ把握できなかった業績指標が、日次あるいはリアルタイムで確認できるようになった。

リアルタイム管理会計ダッシュボードに表示される代表的な指標は以下のとおりである。

- 売上高・利益のリアルタイム進捗（予算対比、前年同期対比）
- 製品別・顧客別の貢献利益リアルタイム集計
- 主要KPIの自動モニタリングとアラート機能
- キャッシュフローのリアルタイム予測
- ROICツリーの構成要素のリアルタイム表示

これらのダッシュボードは、経営会議での報告資料の作成工数を劇的に削減するだけでなく、異常事態の早期発見と迅速な対応を可能にする。例えば、特定の製品ラインの貢献利益率が突然低下した場合、その原因（原材料価格の高騰、歩留まりの悪化、販売価格の下落等）をドリルダウンして即座に特定できる。

### 12-4. 市場規模と将来展望

世界のAI関連管理会計・財務ソフトウェア市場は、急速な成長を遂げている。調査会社の推計によれば、市場規模は2025年の約67億ドル（約1兆円）から2030年には約376億ドル（約5.6兆円）に拡大すると予測されており、年平均成長率（CAGR）は約41%に達する。

この成長を牽引する要因は以下の3点である。

**（1）自動化による効率化ニーズ：** 人手不足が深刻な日本企業において、経理・財務部門の業務自動化は喫緊の課題である。AI-OCRによる請求書の自動読み取り、仕訳の自動起票、異常取引の自動検知などが実用化されている。

**（2）予測精度の向上への期待：** 不確実性の高い経営環境において、AIによる予測分析の精度向上は、意思決定の質を直接的に高める。シナリオ分析やモンテカルロ・シミュレーションをAIが自動的に実行し、複数の将来シナリオとその発生確率を提示するシステムが既に商用化されている。

**（3）データドリブン経営の浸透：** 経営の意思決定をデータに基づいて行うという考え方が、日本企業にも急速に浸透している。データの収集・統合・分析の基盤としてAIプラットフォームを導入する企業が増えている。

### 12-5. 戦略会計とデジタルトランスフォーメーションの融合

戦略会計の未来は、デジタルトランスフォーメーション（DX）との融合にある。本書で解説した各種フレームワークは、いずれもAIやデジタル技術との組み合わせによって、その効果が飛躍的に高まる。

- **原価企画 x AI：** AIによる市場価格予測と原価シミュレーションにより、原価企画のスピードと精度が向上する。設計段階でのコスト見積もりにAIモデルを活用し、数千パターンの設計案から最適なコスト構造を自動探索するシステムが開発されている。

- **ABC x データ自動収集：** IoTセンサーやRFIDタグにより、各活動の実績データがリアルタイムに収集され、ABCの精度が飛躍的に向上する。従来は推定に頼っていたコストドライバーの消費量が、実測値に基づいて正確に把握できるようになる。

- **BSC x BI：** BIツールとの連携により、BSCの4つの視点のKPIがリアルタイムダッシュボードとして可視化され、戦略の実行状況を常時モニタリングできる。

- **CVP分析 x シミュレーション：** AIを活用したモンテカルロ・シミュレーションにより、不確実性を織り込んだCVP分析が可能になる。「損益分岐点を下回る確率」を確率分布として把握し、リスクを定量化できる。

- **TOC x デジタルツイン：** 生産ラインのデジタルツイン（仮想モデル）を構築し、ボトルネックの特定と改善策のシミュレーションを仮想空間上で行うことで、実際の生産ラインを停止させることなく最適化を進められる。

これらの技術革新は、戦略会計の実務家にも新たなスキルセットを要求する。データサイエンスの基礎知識、AIツールの活用能力、そしてデジタル技術と会計知識を橋渡しする能力が、次世代のCFOやコントローラーに求められる必須要件となるだろう。

戦略会計は「数字を経営判断の武器にする」技術であるが、AIとデジタル技術の進化により、その武器の切れ味はかつてないほど鋭くなっている。しかし、最終的に意思決定を行うのは人間であり、数字の背後にある事業の本質を見抜く力、そして倫理的な判断力は、テクノロジーには代替できない。戦略会計の未来は、人間の知恵とテクノロジーの力を最適に組み合わせるところに開かれている。

**本章のまとめ**

本章では、戦略会計の12のフレームワークを体系的に解説した。原価企画からAI活用まで、各手法は企業の競争環境や経営課題に応じて選択・組み合わせて活用することが重要である。

| 章 | フレームワーク | 主な適用場面 |
|----|-------------|------------|
| 第1章 | 戦略会計概論 | 経営と会計の統合的理解 |
| 第2章 | 原価企画 | 新製品開発時のコスト設計 |
| 第3章 | ABC | 間接費の適正配賦、製品別収益性分析 |
| 第4章 | BSC | 戦略実行の管理、業績評価 |
| 第5章 | CVP分析 | 損益分岐点の把握、価格・数量の意思決定 |
| 第6章 | バリューチェーン分析 | コスト構造の競合比較、投資判断 |
| 第7章 | ライフサイクル・コスティング | 製品の総コスト管理、購買判断 |
| 第8章 | TOC | ボトルネック管理、生産最適化 |
| 第9章 | アメーバ経営 | 小集団独立採算、従業員の経営参画 |
| 第10章 | 品質原価計算 | 品質改善の経済的評価 |
| 第11章 | ROIC経営 | 資本効率の改善、事業評価 |

戦略会計の各手法は、それぞれが独立して存在するのではなく、相互に補完し合う関係にある。例えば、原価企画（第2章）ではバリューチェーン分析（第6章）の視点が不可欠であり、ROIC経営（第11章）の改善にはABC（第3章）やTOC（第8章）の手法が有効に機能する。読者には、これらのフレームワークを単なる知識として習得するだけでなく、自社の経営課題に照らし合わせ、実践的に活用することを期待する。
